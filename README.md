# Email OSINT Tool 🔍

This is a simple Python-based OSINT tool that attempts to check for digital footprints across various social media platforms using an email address.

## 🛠️ Features
* Completely free and open-source.
* Easy to use (CLI-based).
* Works directly without any third-party API keys.

## 📱 How to Run in Termux

```bash
pkg install python git -y
git clone [https://github.com/Almashkhan7860/email-osint.git](https://github.com/Almashkhan7860/email-osint.git)
cd email-osint
pip install holehe
python main.py


## 🐧 For Linux (Ubuntu/Debian)
Copy the code below to run it in the Linux terminal:

sudo apt update && sudo apt install python3 python3-pip git -y
git clone [https://github.com/Almashkhan7860/email-osint.git](https://github.com/Almashkhan7860/email-osint.git)
cd email-osint
pip3 install holehe
python3 main.py
